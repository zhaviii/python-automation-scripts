## 📌 Purpose

Collection of Python automation scripts built for real sysadmin tasks.

## Scripts

| Script | Purpose |
|--------|--------|
| firewall_log_analyzer | Parse firewall logs and export to Excel |
| ping_sweep.py | Sweep a subnet and report live hosts |
| service_checker.py | Check if services are running on remote hosts |

See [firewall-log-analyzer](https://github.com/zhaviii/firewall-log-analyzer) for the full version of the log analyzer.
